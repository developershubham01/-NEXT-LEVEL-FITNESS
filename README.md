## 🌟 About The Project

**Next Level Fitness** is a premium gym website built to deliver a modern fitness brand experience.

It includes:

- ⚡ Lightning fast performance with Vite  
- 🎨 Modern UI with Tailwind CSS  
- ✨ Smooth animations using Framer Motion  
- 📱 Fully responsive design  
- 📲 WhatsApp integration for plan booking  
- 🧮 Built-in BMI Calculator  

This project is designed to be production-ready and portfolio-level.

---

## ✨ Key Features

- ✅ Hero section with animated intro  
- ✅ Smooth scrolling navigation  
- ✅ Pricing plans with dynamic WhatsApp redirection  
- ✅ BMI Calculator with real-time results  
- ✅ Contact form section  
- ✅ Google Maps integration  
- ✅ Floating WhatsApp button

⚙️ Installation & Setup
📥 Clone Repository
```bash
git clone https://github.com/developershubham01/-NEXT-LEVEL-FITNESS.git
cd -NEXT-LEVEL-FITNESS
```
📦 Install Dependencies
```bash
npm install
```
▶ Run Development Server
```bash
npm run dev
```
Open in browser:
```bash
👉 http://localhost:5173
```
🏗 Build for Production
```bash
npm run build
```

👀 Preview Production Build
```bash
npm run preview
```
